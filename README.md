## 🚀 Project Description
This Final Project demonstrates an AI-powered Emotion Detection Web Application using IBM Watson NLP.

The application analyzes user input text and detects emotions such as joy, sadness, anger, fear, and disgust.

## 🌟 Features
- Real-time emotion detection
- Interactive UI
- Voice input 🎤
- Chart visualization 📊
- Dark/Light mode 🌙☀️
